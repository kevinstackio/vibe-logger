# VibeLogger

## log session

Saves a snapshot of the current session to `.vibe-logs/YYYY-MM-DD_HH-MM-SS.md`.

1. Run in parallel:
   ```bash
   git log --oneline -10 2>/dev/null
   git diff --stat HEAD 2>/dev/null
   git branch --show-current 2>/dev/null
   date "+%Y-%m-%d_%H-%M-%S"
   ```

2. If the user described what they did, use that as Summary. Otherwise ask once.

3. Write to `.vibe-logs/YYYY-MM-DD_HH-MM-SS.md`:
   ```
   ## {DATE}

   **Project:** {basename of cwd}
   **Branch:** `{BRANCH}`
   **Tool:** Claude Code
   **Summary:** {SUMMARY}

   ### Commits
   {git log}

   ### Files touched
   {git diff --stat}
   ```

4. Ensure `.vibe-logs/README.md` exists (create if missing — see template below).

5. Confirm: `Logged → .vibe-logs/{filename} ✓`

## disable vibe logger

Create `.vibe-logs/.disabled` (empty file). Auto-logging will stop for this project.
Confirm: `VibeLogger disabled for this project. Say "enable vibe logger" to resume.`

## enable vibe logger

Delete `.vibe-logs/.disabled` if it exists. Auto-logging resumes.
Confirm: `VibeLogger enabled ✓`

## .vibe-logs/README.md template

```markdown
# .vibe-logs

Auto-generated by [VibeLogger](https://github.com/kevinstackio/vibe-logger).
Records every Claude Code session automatically.

## Commands

- `log session` — manually log the current session
- `disable vibe logger` — stop auto-logging for this project
- `enable vibe logger` — resume auto-logging for this project
```
